Flex Wrap

Wrap

Primeiro devemos definir a div.pai como display: wrap, já que o flex-wrap é uma propriedade de flex-container, ou seja todos os estilos serão definidos na div.pai.

O flex-wrap define se os elementos irão quebrar wrap ou não nowrap no final do container. Então os valores do flex-wrap são: nowrap que é o valor padrão, wrap ou wrap-reverse que irá quebrar os elementos no final do container de forma inversa.
